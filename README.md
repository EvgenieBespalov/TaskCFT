# TaskCFT
<h1 align="center">Инструкция</h1>
<p>Начальное окно программы, в выделенную область необходимо ввести BIN карты, после чего будет загружена информация о карте и банке</p>
<img src="https://user-images.githubusercontent.com/95974491/209699216-44fedf4b-91c4-42d1-97e1-221015d7183c.png" height="700"/>
<p>После ввода BIN выводится информация о карте и банке. Синим цветом выделены кликабельные элементы:</p>
<li>
  <ol>При нажатии на ссылку сайта будет осуществлен переход на сайт банка в браузере</ol>
  <ol>При нажатии на номер телефона будет осуществлен переход в приложение для осуществления звонков с уже набранным номером банка</ol>
  <ol> При нажатии на название страны будет осуществлен переход в приложении карт с переходом к координатам страны банка</ol>
<li>
  <img src="https://user-images.githubusercontent.com/95974491/209699498-4fa3503e-037b-4009-81fa-d061c2772cfd.png" height="700"/>
<p>Если BIN был не найден, выведется сообщение о неудаче поиска</p>
<img src="https://user-images.githubusercontent.com/95974491/209699557-16bf3e03-068c-4457-8f86-d223c6b58e91.png" height="700"/>
<p>Для просмотра истории поисков необходимо осуществлить свайп вправо</p>
<img src="https://user-images.githubusercontent.com/95974491/209699569-672f5f2e-3983-4b11-8419-8b7c829ff78e.png" height="700"/>
<p>В окне истории поисков выводится список информации о найденных карт с короткой информацией: BIN карты и банк</p>
<img src="https://user-images.githubusercontent.com/95974491/209699621-221ce740-1a3a-411b-9c79-7455c73ba09c.png" height="700"/>
<p>При нажатии на элемент списка выводится подробная информация о найденной карте</p>
<img src="https://user-images.githubusercontent.com/95974491/209699831-81cf308e-7ebf-41f0-8989-8204b22b83b2.png" height="700"/>
